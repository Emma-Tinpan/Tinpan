# PostgrreSQLコンテナを使いこなそう！

## はじめに
PostgreSQLをDockerで使いこなせるようになる！！

## ゴール
下記のことをできるようにすることをゴールとする  
1. PostgreSQLをDockerコンテナで起動する
1. dockerfileでUser、passwordなどの設定済みのコンテナを作る
1. docker-composeでUser、passwordやポートや環境変数など設定したymlファイルから起動できるようにする
1. docker-compose upをしたときにテーブルも完成した状態にできるようになる
1. k8sで使えるmanifestファイルを作る
